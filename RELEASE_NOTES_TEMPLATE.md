### Installation

**For most users:**
1. Download `TorchlightInfiniteTracker.exe` from the Assets section below
2. Run the executable - no installation required!

**System Requirements:**
- Windows 10 or later
- Torchlight: Infinite game installed
- No Python installation required

### First-Time Setup

1. Launch the tracker
2. Start Torchlight: Infinite
3. Click "Initialize Tracker" in the application
4. Follow the on-screen instructions to sort your inventory
5. Start playing!

### Upgrading from Previous Version

Simply download the new version and replace your old executable. Your settings and data files will be preserved.

### Troubleshooting

**Antivirus Warning:**
- Some antivirus programs may flag the executable as suspicious (false positive)
- This is common with PyInstaller-built executables
- You can safely add an exception for this file

**Tracker Not Detecting Game:**
- Make sure Torchlight: Infinite is running
- Ensure the game has logging enabled
- Try restarting both the game and the tracker

**For more help, please open an issue on GitHub.**
